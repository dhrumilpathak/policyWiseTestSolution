----------HOW TO RUN Test -From .Bat file -------------

-> Unzip the folder "GatewayrGroup.Test.Project"  and copy project to C:\
-> Navigate to .\\GatewayrGroup.Test.Project\GatewayrGroup.Test.Project
-> DoubleClick on RunTest.bat

----------HOW TO RUN Test -From Command Prompt  -------------

-> Unzip the folder "GatewayrGroup.Test.Project"  and copy project to C:\
-> Now open CMD and navigate to folder ".\\GatewayrGroup.Test.Project\GatewayrGroup.Test.Project" 

-> Enter "dotnet test" 
-> Wait for the test to execute.
